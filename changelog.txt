## v1.1.3 — 2026-04-13

- Fixed ERG release in stopServo(), pauseServo(), stopPowerErg(): replaced
  malformed opcode 0x08 (Stop, missing required parameter) with opcode 0x11
  (Set Indoor Bike Simulation Parameters) with flat-road values [0x11, 0x00,
  0x00, 0x14, 0x00, 0x28, 0x33] — confirmed from GATT capture of MyWhoosh
  handing back control; transitions KICKR from ERG to slope/simulation mode
  cleanly without a hard stop

## v1.1.2 — 2026-04-13

- Fixed cadence recording: onCPS was resetting lastCadence to null every second
  (readCadenceFromCPS always returns null), wiping values set by FTMS Indoor;
  cadence and speed are now owned exclusively by the FTMS Indoor handler
- Fixed Power ERG not sending any wattage target: FTMS handshake and response
  handler only sent power when servoActive (HR servo flag), never ergActive;
  fixed in ftmsHandshake(), onCPResponse(), and runHeartbeat() in pid.js
- Fixed FTMS Indoor flag parsing: bit-3 (avg cadence) was skipped as 3 bytes
  (should be 2), bit-4 (total distance) skipped as 2 bytes (should be 3),
  bit-5 (resistance level) was not skipped at all
- Added speed capture from FTMS Indoor (always present at offset 2, 0.01 km/h)
  displayed live as amber cv-block; added to session samples and FIT records
- Added distance calculation (speed integrated over time); shown in stats table
  and written to FIT session message (total_distance, avg_speed fields)

## v1.1.1 — 2026-04-13

- Reverted font size doubling (v1.1.0 item removed); all sizes restored to original
- Fixed all buttons non-functional: duplicate `let pendingServoPowerSend` in pid.js
  caused Chrome to throw SyntaxError on pid.js load, leaving every button except
  the two lambda-wrapped Connect buttons unbound; removed duplicate declaration and
  converted all wireButtons callbacks to arrow functions
- Fixed FIT export rejected by intervals.icu ("File CRC failed"): CRC table used
  wrong polynomial and wrong XOR sequence; replaced with Garmin SDK table and
  FitCrc_Get16 algorithm verbatim
- Fixed cadence not captured: FTMS Indoor was skipped when CPS succeeded; now always
  subscribed so cadence arrives via FTMS regardless of power source
- Fixed download buttons (CSV, FIT) permanently disabled: added enable in onSessionStopped
- Added .gitignore covering session logs, FIT/CSV exports, and scratch files

## v1.1.0 — 2026-04-13

- Default values updated: RestHR 43, MaxHR 173, FTP 250
- Doubled font sizes across all data displays and settings panels

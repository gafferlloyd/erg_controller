## v1.1.9 — 2026-04-14

- KICKR BIKE SHIFT BLE protocol fully decoded from GATT capture (gatt_explorer
  as sole connected device): gear state on char a026e03a (service a026ee0d),
  button press on char a026e03c; no app-layer init write required — CCCD
  subscription alone delivers notifications
- Gear state: 6-byte CHANGE packet (type 01) and 12-byte STATE heartbeat (type 07);
  rear index 0 = 34T largest cog (easiest), index 11 = 11T (hardest); chainring
  index 0 = 36T inner, 1 = 52T outer; Shimano Ultegra 2×12
- Button press: 3-byte format, 16-bit big-endian bitmask (10 buttons mapped),
  bit 7 of byte 2 = press vs. release
- gatt_explorer.html: OPT_SERVICES expanded to include ee01–ee0d; KNOWN labels
  and parseNotification() decoders added for gear state and button press chars
- _kickr_shift_guide.html: full protocol reference document created

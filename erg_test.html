Step 4 — Handshake Strategy
A · Minimal: [0x00] Request Control only — the minimum the spec requires before writing power.
B · Full: [0x00] Request → [0x01] Reset → [0x00] Request again → [0x07] Start — mirrors exactly what Zwift sends.
C · Wahoo: Writes to Wahoo's proprietary control characteristic using their known opcode format.
Watch the log for [RX] indication responses — 0x8001 = success.
